## myblog

## 기능 목록
1. 게시글 생성
   + 메인 페이지에서 작성 페이지로 이동
   + 게시글 생성 시 게시글 페이지로 이동
2. 게시글 목록 조회
   + 메인 페이지에서 게시글 조회
3. 게시글 조회
   + 메인 페이지에서 게시글 클릭 시 해당 게시글로 이동
4. 게시글 수정
   + 게시글 페이지에서 수정 버튼 클릭 시 수정 페이지로 이동
   + 수정 버튼 클릭 시 해당 게시글의 정보 가져오기
   + 수정 완료 후 게시글 페이지로 이동
5. 게시글 삭제
   + 삭제 버튼 클릭 시 게시글 삭제
   + 삭제 완료 후 메인 페이지로 이동
    
## 제약 조건
+ HTML 중복제거
+ 정적 파일 수정 시 재시작 하지 않고 변경사항 반영하기
+ class 파일 수정 시 자동으로 재시작 하기
