## myblog

## 기능 목록
1. 게시글 생성
   + 메인 페이지에서 작성 페이지로 이동
   + 게시글 생성 시 게시글 페이지로 이동
2. 게시글 목록 조회
   + 메인 페이지에서 게시글 조회
3. 게시글 조회
   + 메인 페이지에서 게시글 클릭 시 해당 게시글로 이동
4. 게시글 수정
   + 게시글 페이지에서 수정 버튼 클릭 시 수정 페이지로 이동
   + 수정 버튼 클릭 시 해당 게시글의 정보 가져오기
   + 수정 완료 후 게시글 페이지로 이동
5. 게시글 삭제
   + 삭제 버튼 클릭 시 게시글 삭제
   + 삭제 완료 후 메인 페이지로 이동
    
6. 회원 가입
   + 이름은 2자 이상 10자 미만
   + 비밀번호는 8자 이상 대소문자 특수문자 숫자 포함
   + 이메일은 중복 불가
7. 로그인
   + 로그인 후에 블로그 서비스 이용가능
   + "/" 루트 페이지는 예외
8. 회원정보 수정
   + 이름과 이메일 수정 가능
9. 회원 탈퇴
   + 로그인 이후에 회원 탈퇴 가능

## 제약 조건
+ Interceptor를 이용하여 로그인 구현
